Creating a Website with HTML and CSS: Mobile Legends Heroes

  • Project URL: github.com/jamelle01/MLBB-html-css-Project
  • Project demo: Live Demo

During my college days, I worked on a project to create a website using HTML and CSS. The website was focused on Mobile Legends heroes, a popular mobile game, and aimed to showcase information about each hero in an organized and visually appealing way.

Using HTML and CSS, I designed and implemented a layout that highlighted each hero's unique attributes, including their name, role, and abilities. I also incorporated images and animations to make the website more engaging and interactive for users.

Overall, this project allowed me to gain valuable experience in web development and design, as well as develop my skills in HTML and CSS. It was a challenging and rewarding experience, and I'm proud of what I was able to accomplish.
